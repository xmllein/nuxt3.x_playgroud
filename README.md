## Nuxt 3 playground

- nuxt3.x 练习

## 安装依赖

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## 启动开发环境

服务端口 on http://localhost:3000

```bash
npm run dev
```

## 生产环境打包

打包生产文件:

```bash
npm run build
```

本地预览打包后的文件:

```bash
npm run preview
```
